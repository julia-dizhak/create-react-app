# create-react-app
Create React apps by egghead course start learning react

## lesson1: use [create-react-repo](https://github.com/facebookincubator/create-react-app) to setup a simple react app
React requires some initial setup before we can get going with the app.
```
yarn global add create-react-app
create-react-app my-app
cd my-app
yarn start
```
